# Kaş Limoncello

Premium multilingual limoncello landing page built with Astro, Tailwind CSS, and GSAP-powered motion.

## Live Site

https://7limonikas7.github.io/limonchelooa/

## Repository

https://github.com/7LimoniKAS7/limonchelooa

## Routes

- `/tr/` Turkish
- `/en/` English
- `/ru/` Russian

## Development

```bash
npm install
npm run dev
```

## Production

```bash
npm run check
npm run build
```

## Security

- Silent honeypot and submit-timing checks protect the WhatsApp order flows from basic bots.
- Short session cooldowns reduce repeat automated submissions.
- Production pages include a restrictive Content Security Policy, referrer policy, and limited browser permissions.
- Security contact metadata is available at `/.well-known/security.txt`.
